## ADDED Requirements

### Requirement: 프로필 사진 관리
시스템은 인증 사용자가 본인의 프로필 사진을 등록·교체·삭제할 수 있게 MUST 해야 한다.

#### Scenario: 프로필 사진 등록
- **WHEN** 사용자가 5MB 이하 JPEG, PNG 또는 WebP 사진을 선택하고 프로필을 저장한다
- **THEN** 시스템은 본인 전용 Storage 경로에 사진을 저장하고 프로필에 연결한다

#### Scenario: 프로필 사진 교체
- **WHEN** 기존 사진이 있는 사용자가 새 사진으로 프로필을 저장한다
- **THEN** 시스템은 새 사진 연결이 성공한 뒤 이전 사진 객체를 삭제한다

#### Scenario: 프로필 사진 삭제
- **WHEN** 사용자가 사진 삭제를 선택하고 프로필을 저장한다
- **THEN** 시스템은 프로필의 사진 연결과 기존 객체를 제거하고 기본 글자 아바타를 표시한다

#### Scenario: 지원하지 않는 사진
- **WHEN** 사용자가 허용되지 않은 형식 또는 5MB를 초과하는 파일을 선택한다
- **THEN** 시스템은 업로드하지 않고 형식 또는 크기 오류를 표시한다

### Requirement: 프로필 이미지 접근 제어
시스템은 프로필 사진을 채팅 참여자에게 표시하되 객체 생성·수정·삭제는 경로 소유자에게만 MUST 허용해야 한다.

#### Scenario: 다른 사용자 경로 쓰기
- **WHEN** 인증 사용자가 다른 사용자 ID의 Storage 경로에 사진을 쓰거나 삭제하려 한다
- **THEN** Storage 정책은 요청을 거부한다

#### Scenario: 채팅 메시지 아바타 조회
- **WHEN** 사용자가 메시지 내역을 조회하거나 새 메시지를 실시간 수신한다
- **THEN** 시스템은 발신자의 현재 프로필 사진 또는 기본 글자 아바타를 표시한다

### Requirement: 프로필 사진 크게 보기
시스템은 등록된 프로필 사진을 아바타에서 전체 화면으로 MUST 열 수 있게 해야 한다.

#### Scenario: 아바타 선택
- **WHEN** 사용자가 사진이 등록된 채팅 아바타 또는 본인 아바타를 선택한다
- **THEN** 시스템은 어두운 전체 화면 레이어에 원본 비율로 사진과 닫기 동작을 표시한다

#### Scenario: 사진 확대와 이동
- **WHEN** 사용자가 열린 프로필 사진을 확대하거나 드래그한다
- **THEN** 시스템은 사진 확대·축소와 화면 내 이동을 지원한다
